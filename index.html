Wheel-Way

Wheel-Way is an app designed for people who use wheelchairs to get from point A to point B. This app has an intuitive user interface with Google Maps. I worked on the map in this project.

When the map displays, the user can click a spot on the map. The user will then have to click the spot on the map again. Once clicked, a form will display with options that identify an obstruction in the path. Save the change, and the icon will be saved on the map in real time for other users to see!

Firebase was used to store the icon data.

TeeTime

TeeTime is an idea I've had that comes from by background in golf. This app is designed for private clubs to use as a tee sheet. The difference here is that the members and guests can book their teetimes in real time! That will help save the Golf Pro a lot of time to do other things around the shop.

I worked on the front-end of this app. I designed with more of a traditional look that both golf pros and golfers are familiar with. This is designed to effectively book a tee-time with minimal effort. Golf pros will be able to access the admin side of the tee sheet. There they can manipulate the times available for golfers to play.

PrepConnect

PrepConnect is a Full-Stack Application created with the MERN Stack. This is designed to help aspiring web developers land their first developer job. I worked on this with a team of four. I worked on the Font-End using React.js.

PrepConnect is designed to take the focus off of the jobseeker's resume and more on their skill. Here a jobseeker will create a profile by rating their skills in certain coding languages.

A Recruiter will go through a very similar process when posting a job. There are two added steps - Set minimum compatibility, and Verify a skill with a question. Then the job posting and jobseeker profile will be matched.

Then, a jobseeker will see a job posting, and can actually take the application immediately. Select 'view appliction' to start the application process. Use editors to answer recruiters questions!
